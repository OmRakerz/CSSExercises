Задание №1

Пусть главная ось горизонтальна. Куда будет направлена поперечная ось?

Если главная ось горизонтальная, то поперечная ось будет направлена сверху вниз.


Задание №2

Пусть главная ось вертикальна. Куда будет направлена поперечная ось?

Если главная ось вертикальная, то поперечная ось будет направлена слева направо.


Задание №3

Пусть поперечная ось направлена направо. Куда в таком случае может быть направлена главная ось?

Если поперечная ось направлена направо, то главная ось может быть направлена или сверху вниз, или снизу вверх.


Задание №4

Пусть поперечная ось направлена вниз. Куда в таком случае может быть направлена главная ось?

Если поперечная ось направлена вниз, то главная ось может быть направлена или справа налево, или слева направо.


Задание №5

Может ли поперечная ось быть направлена справа налево?

Нет, т.к. поперечная ось может быть направлена только сверху вниз или слева направо.


Задание №6

Может ли поперечная ось быть направлена снизу вверх?

Нет, т.к. поперечная ось может быть направлена только сверху вниз или слева направо.


Задание №7

Пусть свойство flex-direction имеет значение row. Куда будет направлена главная ось? Куда будет направлена поперечная ось?

Значение flex-direction: row (по умолчанию) => главная ось направлена слева направо , следовательно, поперечная ось будет направлена сверху вниз.


Задание №8

Пусть свойство flex-direction имеет значение column. Куда будет направлена главная ось? Куда будет направлена поперечная ось?

Главная ось будет направлена сверху вниз => поперечная ось будет направлена слева направо.


Задание №9

Пусть свойство flex-direction имеет значение row-reverse. Куда будет направлена главная ось? Куда будет направлена поперечная ось?

Главная ось будет направлена справа налево => поперечная ось будет направлена сверху вниз.


Задание №10

Пусть свойство flex-direction имеет значение column-reverse. Куда будет направлена главная ось? Куда будет направлена поперечная ось?

Главная ось будет направлена снизу вверх => поперечная ось будет направлена слева направо.
